📘 Descripción general del sistema
Proyecto con distintas clases para mantener orden y disgregación para las distintas personas y entidades que se relacionan con la empresa. El proyecto incluye la clase main para su ejecución, además, incluye una clase 
padre "Persona" y las subclases: Cliente, Empleado y Proveedor. Finalmente tiene la clase Datos y la clase CorreoInvalidoException para controlar errores de ingresos de correo.
La finalidad del proyecto es mostrar la capacidad de crear proyecos con herencias, principios de encapsulamiento, composición y reutilización de clases.


🧱 Estructura general del proyecto (Ubicado en la carpeta scr/main/java)

📁 src/

├── app/         # Clase principal con el método main

├── model/       # Clases de dominio (Persona, Datos, Cliente, Empleado, Proveedor, CorreoInvalidoException)
